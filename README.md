# XOLERIC (Veee)

A polished book reader app with 50 books, a horizontal progress scale for paragraph navigation, and TTS playback.

## Features
- 50 books with category filtering
- Horizontal progress scale (tick marks per paragraph)
- TTS (text-to-speech) with play/pause
- Font size zoom panel
- Search and category filters
- Last-read book continuation banner
- Smooth paragraph transitions with active highlighting

## How to open
Open `index.html` in any modern browser.
